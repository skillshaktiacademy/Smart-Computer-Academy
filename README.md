# Smart Computer Academy

A professional educational platform featuring a separate Frontend and Backend architecture.

## Repository Structure

- **Frontend/**: Built using React and Vite, delivering a fast, interactive, and beautiful user experience.
- **Backend/**: Built using Node.js and Express to handle administrative API logic, database connections, and business rules.

## Setup Instructions

### Prerequisites
- Node.js installed on your machine.
- MongoDB installed locally or a remote connection string.

### Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/skillshaktiacademy/Smart-Computer-Academy.git
   cd Smart-Computer-Academy
   ```

2. **Frontend Setup**:
   ```bash
   cd Frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**:
   ```bash
   cd Backend
   npm install
   npm run dev
   ```
